# Ressources vidéos & documents — enrichir les 5 leçons « 000-Initiation »

Objectif : **s’inspirer** (méthodes, chiffres, angles) pour étoffer les scripts **oraux en darija** ; ne pas recopier des contenus protégés. Toujours **vérifier la licence** avant réutilisation d’extraits.

### Règle — langue des sources vs langue du produit

- **Méthodes** (stress, organisation, recherche d’info, structuration d’un projet, entretien, etc.) : les sources peuvent être en **n’importe quelle langue** ; ce qui compte, c’est de **extraire la bonne pratique** et de l’**expliquer en darija** dans nos vidéos/leçons.  
- **Contexte marocain** (marché du travail, recrutement, diplômes, chiffres) : **vigilance** — privilégier **données et réalités locales** (ex. HCP, presse marocaine) et **adapter** tout contenu importé (France, international).

## Méthode de veille (rapide)

| Canal | Requêtes types |
|--------|----------------|
| YouTube | `recherche stage ingénieur`, `orientation parcours professionnel`, `soft skills entreprise`, `entretien embauche méthode STAR` |
| Rapports | `HCP Maroc marché du travail`, `adéquation formation emploi Maroc` |
| Universités (FR) | `SCUIO recherche stage`, `méthodologie insertion professionnelle podcast` |
| LinkedIn / articles | `attentes recruteurs Maroc`, `compétences techniques vs comportementales` |

**Critères de qualité** : source identifiable (institution, date), cohérence avec le public **étudiant marocain**, pas de promesse magique d’emploi.

## Par leçon (thèmes à creuser)

### Leçon 1 — Trailer parcours (émotions + étapes bac → emploi)

- **Données Maroc** : [HCP — Adéquation formation / emploi](https://www.hcp.ma/L-adequation-entre-formation-et-emploi-au-Maroc-Rapport-complet_a2170.html), [Études marché du travail (HCP)](https://www.hcp.ma/Etudes-Marche-du-travail_r651.html) — pour **une phrase choc** factuelle (chômage jeunes, déclassement) traduite en message humain en darija.
- **Vidéos orientation (ton pédagogique)** : chaînes type **Onisep TV** (parcours, recherche de stage) — [ex. école d’ingénieurs / recherche de stage](https://oniseptv.onisep.fr/video/ecole-dingenieurs-a-la-recherche-dun-stage) (adapter le contexte au Maroc).

### Leçon 2 — Questions par étape (métier vs domaine, marché, stage → CDI)

- **Projet professionnel / compétences** : MOOC type [Construire son projet professionnel (OpenClassrooms)](https://mooc-francophone.com/cours/construire-son-projet-professionnel/) — structuration des **questions** que vous posez déjà dans le script.
- **Méthode recherche stage** : [SCUIO-IP Montpellier — vidéos méthodologie stage-emploi](https://pod.univ-montp3.fr/scuio-ip-information-orientation-insertion-professionnelle/methodologie-de-recherche-stage-emploi/) — pour enrichir *sans* changer votre ligne éditoriale proM7.

### Leçon 3 — Domaine, métier, personnalité, témoignages

- **Ingénieur / métiers** : [FUN MOOC — Devenir ingénieur, c’est possible](https://www.fun-mooc.fr/fr/cours/devenir-ingenieur-cest-possible/) — témoignages et **variété de trajectoires** (utile pour cadrer vos histoires Abdelbarie / Zaq / Abdeljalil).
- **Soft skills & différenciation** : rechercher `employabilité compétences comportementales étudiant` + filtres **durée < 15 min** pour extraits utilisables en storyboard.

### Leçon 4 — Méthodologie proM7 (darija + FR/EN pro, réalité entreprise)

- **Communication pro / entreprise** : vidéos courtes sur `réunion équipe`, `gestion du temps projet`, `priorisation` (vocabulaire à gloser en darija dans vos scripts).
- **E-learning & interaction** : s’inspirer des formats **micro-learning** (une idée = une vidéo) plutôt que de longs cours.

### Leçon 5 — Bénéfices du parcours (honnêteté, outils, « langue employeur »)

- **Entretien / CV** (méthodes reproductibles) : [L’Étudiant — préparer entretien](https://www.letudiant.fr/jobsstages/lettres-de-motivation_1/comment-preparer-un-entretien-d-embauche.html), [CVmaker — entretien](https://www.cvmaker.fr/blog/entretien-embauche/preparation-entretien-embauche) — pour la **méthode** (ex. STAR), reformulée en darija.
- **CV vidéo / entretien vidéo** : [France Travail — CV vidéo](https://www.francetravail.fr/candidat/vos-recherches/preparer-votre-candidature/cv-lettre-de-motivation-e-mail/cv-video-comment-vous-demarquer.html) — utile si vous ajoutez un module plus tard.

## Sources Maroc « institutionnelles » (contexte)

- **ONOUSC** : [onousc.ma](https://www.onousc.ma/) — surtout logement / vie étudiante ; [vidéothèque](https://www.onousc.ma/videotheque) — peu orientée « emploi » mais utile pour **ancrage universitaire**.
- **Stage.ma / offres** : utile pour **observer** intitulés de postes et secteurs (veille marché), pas comme script prêt à l’emploi.
