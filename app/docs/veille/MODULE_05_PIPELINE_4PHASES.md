Phases 1–3 synthétisées ici ; Phase 4 = `tools/module05_rich_content.py` + `python tools/format_module05_scripts.py`.

**Style oral (aligné M4)** : **تلخيص** / **أغلاط** / **الجسر** en **phrases reliées** (darija + connecteurs), pas listes télégraphiques seules.
