**Cadres nommés (vulgarisation script)** : parcours M0–M7 ; **جدول** intérêt × marché ; protocole **4 étapes** avant question ; **Pomodoro** / **time blocking** ; **SMART** / **SWOT** ; **HCP** pour lecture actualisée du marché MA.
