# Modèle — citations pour les scripts (original + darija)

Chaque bloc ci-dessous est un **exemple de structure**. L’humain ou l’agent peut ajouter des entrées ; dans le script final, **ne pas** déformer le sens de l’original.

---

## Entrée type

- **Source** : (livre, article, intervenant, URL — pas de contenu protégé sans droit clair)
- **Langue originale** : FR / EN
- **Citation originale** : « … »
- **Darija (oral script)** : … (reformulation naturelle, ton proM7)
- **Usage suggéré** : (ex. accroche début séquence, leçon M4 L35, rappel M1…)
- **Statut** : `nouvelle` | `intégrée` (si intégrée : pointer le fichier `moduleNN_rich_content.py` et la leçon)

---

## Exemple factice (à supprimer quand de vraies citations sont ajoutées)

- **Source** : Exemple pédagogique interne
- **Langue originale** : EN
- **Citation originale** : « Clarity is kindness in a professional setting. »
- **Darija (oral script)** : « الوضوح فالخدمة هو نوع من اللطف مع الناس — مشي ترف، هو احترام للوقت ديال الزميل. »
- **Usage suggéré** : M3, séquence sur communication / feedback
- **Statut** : `nouvelle`
