## Idées clés (proM7)

- **Hard vs soft** : les **compétences techniques** (métier) vs les **compétences transversales** — surtout la capacité à **travailler avec les autres** ; utile pour le CV, l’entretien et l’**auto-positionnement** (sans sur-promesse emploi).
- **Constat pédagogique** : beaucoup de profils **forts techniquement** mais **relationnel sous-développé** — frein de carrière fréquent (angle motivation pour le parcours).
- **1 — Leadership (sans titre)** : faire avancer un groupe **de A à B** (vision, cohésion, coaching) **avant** d’avoir le mot « manager » sur la fiche — exemple type **contributor** promu grâce au comportement.
- **2 — Communication** : clarté **orale et écrite** (réunions, mails structurés) ; **non-verbal** (regard, visage, posture) qui peut **aider ou saboter** ; niveau avancé : prise de parole, **récit**, argumentation.
- **3 — Relationnel** : **écoute active**, attitude positive, **empathie**, humour léger si contexte ; **assertivité** (dire les choses sans agresser) ; **diplomatie** au désaccord ; donner / recevoir un retour avec **calme**.
- **4 — Éthique de travail / motivation** : fiabilité, **initiative**, respect des délais, assiduité — ce que les recruteurs testent souvent via références et signaux de **micromanagement évité**.
- **5 — Travail en équipe** : collaboration **interprofils** et **hors silo** ; vision **collective** du résultat vs pure individualité.
- **6 — Résolution de problèmes** : enchaînement d’**obstacles** ; **pensée critique**, décisions **argumentées**, méthode + créativité + **débrouillardise** (pas seulement la première idée).
- **7 — Flexibilité / adaptabilité** : accepter le **changement**, l’**incertitude**, les retours ; rester « **facile à travailler** » sous pression — antidote à la **rigidité** qui bloque les projets.
- **8 — Conflits / négociation** : éviter d’être **source de drame** (logique **gagnant-perdant** toxique) ; chercher le **terrain d’entente**, faire sentir les gens **compris**, viser des **sorties gagnant-gagnant**.
