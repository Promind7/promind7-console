## Idées clés (proM7)

- **Définition minimale** : une activité propose **biens ou services** à des **clients** qui **paient** ; pour **durer**, les **recettes** doivent en général **dépasser les charges** (sinon pertes / risque d’arrêt) — schéma pédagogique, pas cours de compta juridique.
- **Même logique pour un projet associatif** : viser **revenus > dépenses** pour la **pérennité** (la source parle de « community project » géré comme une structure viable).
- **Trois rôles internes classiques** : **propriétaires / actionnaires** (capital, équipement, vision ; parfois **conseil d’administration** si beaucoup d’actionnaires), **managers** (priorités du jour, répartition des tâches, **qualité**, **planification**), **travailleurs** (production, vente, **comptabilité**, etc.).
- **Rémunérations** (simplifié dans la vidéo) : **salaire / wage** pour exécution ; **salaire mensuel** type cadre pour manager ; **dividendes** tirés des **bénéfices** pour les propriétaires — **non garantis** si pas de profit ; un même individu peut cumuler **rôle opérationnel + rémunération** correspondante.
- **Risques** : actionnaires peuvent **perdre** leur mise ; manager peut être **limogé** si résultats insuffisants ; salarié peut **perdre son poste** si l’activité ferme ; **sécurité** au travail (angle humain).
- **Alignement d’intérêts** : qualité → meilleure **fidélisation** ; clients **contents** → rentrées d’argent plus stables — utile pour expliquer **pourquoi** l’entreprise valorise la **qualité de service** et le **travail d’équipe** (lien **soft skills**).
- **Clients = partie prenante** : sans acheteurs / usagers payants (ou financeurs), pas d’**activité économique** au sens classique — pertinent pour la **veille entreprise** avant entretien (**qui paie ? qui décide ?**).
- **Chiffre « 10× » coût acquisition vs fidélisation** : formule **marketing** souvent citée — en proM7, si tu l’utilises en darija, présenter comme **ordre d’idée** ou **éviter** sans source chiffrée locale.
