## Idées clés (proM7)

**Livraison (delivery)**  
- **Pauses** : créer du « blanc » entre les idées pour qu’elles **s’impriment** ; souvent perçu comme plus **assuré**, pas moins.  
- **Ralentir sur l’essentiel** : varier le **rythme** plutôt que tout dire **vite** — le ralentissement **marque** les points importants (comme surligner une phrase, pas toute la page).  
- **Phrases déclaratives** : remplacer le **flou** (*maybe*, *kind of*, tours longs) par des **phrases courtes complètes** — réduit le **radotage** et le côté « incertain ».

**Voix (vocal)**  
- **Échauffement** léger avant un **entretien** ou une **présentation** (ex. exercices type **lip trills**) — à faire **hors** salle d’attente / hors visio ouvertes (comme le dit la source).  
- **Volume sain** : parler assez fort pour **remplir l’espace** sans **crier** — énergie et **présence** auditive.  
- **Respiration** : la source oppose bouche / nez pour **confort vocal** ; en proM7, rester sur l’idée générale « **souffle** et **hydratation** » et **éviter** les chiffres type « +20 % d’oxygène » sans source **médicale**.

**Cognition**  
- **Une pensée à la fois** : enchaîner **idée complète → pause → idée suivante** plutôt que **fragmenter** en milieu de phrase.  
- **Cadre CCC** (*Context* → *Core* → *Connect*) : **contexte**, **message central unique**, **pourquoi ça concerne l’interlocuteur** — utile pour **email**, **réunion**, **entretien**.  
- **Analogies** : si l’autre ne suit pas, **changer de formulation** (image du quotidien) plutôt que **répéter plus fort** le jargon.
