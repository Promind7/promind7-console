## Transcript (texte)

Je rappelle qu'on stage si une période durant laquelle un étudiant vient en quelque sorte

découvrir la vie professionnelle au sein d'autant d'entreprise pour mettre en pratique

ses connaissances théoriques et développer ses compétences.

Ce qu'il faut bien noter c'est que cette expérience-là elle doit être formalisée par

une convention de stage qui est très partie de, c'est-à-dire qu'elle est entre ton

entreprise, l'établissement d'enseignement et le stageur lui-même.

Et c'est le code de l'éducation et non le code du travail qui n'est pas à régir en quelque

sorte cette convention-là.
