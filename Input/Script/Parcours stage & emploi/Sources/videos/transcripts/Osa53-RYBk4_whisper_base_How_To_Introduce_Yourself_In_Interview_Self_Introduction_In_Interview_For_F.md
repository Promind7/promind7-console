## Idées clés (proM7)

- **Objectif de l’intro** : le recruteur teste **adéquation poste + entreprise**, **clarté**, **confiance** — pas un récit de vie privée.
- **Pourquoi parler si le CV est là** : entendre **comment** le candidat se présente et priorise l’information.
- **Structure proposée** : **salutation** (éventuellement remerciement) → **identité** (nom, ville si utile) → **parcours** : **école / projets** (jeune) ou **poste actuel + missions + faits** (expérience) → **ligne** hobbies / activités **si** ça humanise sans diluer → **fermeture** : **motivation** + lien **fiche de poste** + compétences clés.
- **Réalisations** : exemples **vérifiables**, ton **enthousiaste** sans arrogance ; adjectifs type *team player* **épaulés** par un mini-fait.
- **Durée cible** : **environ une minute** ; noter les **points clés** puis parler **naturellement** (éviter la récitation mot à mot).
- **Sous pression** : petites erreurs de langue possibles si le **fond** est clair ; **regard** et **assurance** prioritaires sur la perfection grammaticale orale.
