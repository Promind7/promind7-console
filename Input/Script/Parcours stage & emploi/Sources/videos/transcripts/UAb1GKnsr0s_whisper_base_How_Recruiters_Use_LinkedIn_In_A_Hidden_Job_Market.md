## Idées clés (proM7)

- **Marché « moins annoncé »** : en période de **tension** (chômage, restructurations), certaines entreprises **réduisent** les annonces **grand public** tout en **recrutant encore** — pistes : **page carrières** du site, **veille** ciblée, **réseau**. La narration **US** de la vidéo est **datée** ; le **principe** (moins de visibilité agrégée ≠ zéro embauche) reste **pédagogique** avec **prudence** sur le contexte local.
- **Recruteur chercheur actif** : plutôt que trier des **milliers** de candidatures spontanées, l’orateur décrit un mode où le recruteur **interroge** LinkedIn (outils type **Recruiter**) pour constituer une **liste courte** puis **contacte** quelques profils — intérêt du **profil optimisé** pour la **découvrabilité**, **sans** promesse « ils viendront tous à toi ».
- **Analogie « pages jaunes »** : le profil = **fiche** dans un **annuaire** professionnel ; ce que tu **écris** et **où** tu le places influence les **requêtes** pour lesquelles tu **apparais**.
- **Fonctionnement type moteur de recherche** : beaucoup de profils → la plateforme **classe** selon **pertinence** par rapport aux **filtres** et **mots-clés** du recruteur — renvoie aux bonnes pratiques **titre**, **à propos**, **expériences**, **compétences** (vocabulaire **aligné** sur les postes visés).
- **Test de survol** : une fois dans les résultats, le recruteur **parcourt vite** le profil ; il faut un **premier écran** **clair** (qui je suis, pour quoi je suis pertinent) avant le détail.
- **Juste mesure d’information** : trop **vide** = manque de **crédibilité** ; trop **dense** ou **visuellement chargé** = **fatigue** du lecteur ; viser assez pour **donner envie d’écrire** ou d’appeler, pas tout raconter — proche d’un **message marketing** équilibré.
