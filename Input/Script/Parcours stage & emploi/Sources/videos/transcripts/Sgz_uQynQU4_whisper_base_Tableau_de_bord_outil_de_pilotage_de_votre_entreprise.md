## Idées clés (proM7)

- **Tableau de bord** (sens pro) : outil de **pilotage** et d’**aide à la décision** au-delà du seul **bilan** ou compte annuel — utile pour **anticiper** (**trésorerie**, **recrutement**, **investissement**, montée en charge) et suivre l’**atteinte d’objectifs** chiffrés **réalistes**.
- **Indicateurs financiers** cités : **évolution du chiffre d’affaires** et **saisonnalité** ; **panier moyen** ; **trésorerie** ; **créances clients** / délais d’**encaissement** — vocabulaire à maîtriser en **veille entreprise** ou **stage** en PME.
- **Indicateurs non financiers** : **origine** des **contacts** / prospects ; **taux de transformation** (lien **performance commerciale**) — le pilotage n’est pas que « compta ».
- **Lecture** : comparer **prévu** vs **réalisé**, comprendre les **écarts** (ex. événement externe évoqué : perturbation **transport**) — culture **analyse** plutôt que tableau décoratif.
- **Exemples** (illustrations vidéo) : **CA** détaillé (panier, fréquentation…) ; **pipeline** commercial vs **factures** pour voir où se perd la **conversion** ; **canaux** d’acquisition clients — idées **pédagogiques**, pas chiffres à **recopier**.
- **Bonnes pratiques** : tableau **synthétique** et **lisible** ; **limiter** le nombre d’indicateurs ; éviter le **trop** de détail par indicateur pour garder une **lecture** rapide.
- **Appui** : **expert-comptable**, **banquier** ou autres **professionnels** qui connaissent l’activité — choix des **KPI** pertinents (surtout **TPE/PME**).
- **Statistique d’ouverture** (beaucoup d’**échecs** à 5 ans, gestion) : **contexte** souvent **français** / média — **ne pas** citer le **%** dans un script **sans source primaire** vérifiée.
- **Fin** : renvoi **sites** (Inextens, Point C, BFM) — **promotion** à **filtrer** dans tout usage **proM7**.

**Lien parcours** : **M2** (lire l’entreprise, **finance** de base) ; **M6** (**KPI**, performance) ; **M4** stage en **PME** : comprendre **pourquoi** le dirigeant suit certains **chiffres**.
