# Dossier `concepts/` — tes documents

Ce sous-dossier est prévu pour **tes** fichiers sources (`.docx`, images, etc.) sur les thèmes type pensée critique, entretien, négociation, prise de parole, citations, etc.

- Exemple : dossier **critical thinking** (ou équivalent) — **déjà pris en compte** dans l’enrichissement des leçons (voir entrée du **`../used_sources_registry.md`** et passages « تفكير نقدي » / *critical thinking* dans `tools/module00_rich_content.py`, `module01_rich_content.py`, `module03_rich_content.py`, + rappels M2).
- **À ne pas confondre** : les **analyses / tableaux rédigés par l’agent** vont sous `reports/` (`Sources_concepts_analyse_agent.md`, dispatch, etc.), **pas** ici.
- Si le dossier avait été vidé par erreur : remets-y tes fichiers depuis une **sauvegarde** ou la **Corbeille** Windows si les originaux y sont encore.

Un extrait texte issu d’anciens `.docx` (si tu les avais déjà traités une fois) peut subsister dans le dépôt : `reports/sources_concepts_extract.md` — ce n’est pas un remplacement des binaires.

**Quand tu ajoutes de nouveaux fichiers ici** : après intégration validée dans les `moduleNN_rich_content.py`, ajoute une ligne datée dans `../used_sources_registry.md` (nom de fichier ou sous-dossier précis).
